# AlignPath Nexus

Welcome to AlignPath Nexus! This project is a modern web application built with React, TypeScript, and Vite.

## Project info

**URL**: https://alignpath-nexus.lovable.app/

## Features

- Modern, responsive User Interface
- Fast development environment powered by Vite
- Fully typed with TypeScript
- Styled using Tailwind CSS and shadcn-ui

## Getting Started

### Prerequisites

The only requirement is having Node.js & npm installed. We recommend installing Node via [nvm (Node Version Manager)](https://github.com/nvm-sh/nvm#installing-and-updating).

### Installation and Setup

Follow these steps to get the project running locally:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <index.html>

# Step 3: Install the necessary dependencies
npm i

# Step 4: Start the development server
npm run dev
```

Your development server will start with auto-reloading and an instant preview, typically at `http://localhost:8080`.

## Scripts

- `npm run dev` - Starts the development server at `localhost:8080`
- `npm run build` - Builds the application for production
- `npm run preview` - Locally preview the production build
- `npm run lint` - Runs ESLint to identify and report on patterns in your code
